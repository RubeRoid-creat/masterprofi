# MLM Dashboard Screen

Экран визуализации MLM сети с иерархическим деревом, статистикой, графиками и всеми инструментами для управления сетью.

## Особенности

✅ **Hierarchical Tree View** - Иерархическое дерево сети с раскрывающимися узлами  
✅ **Team Statistics** - Статистика команды (размер, активные участники, заработок)  
✅ **Commission Breakdown** - Разбивка комиссий по уровням  
✅ **Recruitment Progress** - Прогресс рекрутинга с индикаторами  
✅ **Leaderboard** - Таблица лидеров топ-исполнителей  
✅ **Monthly Bonus Tracker** - Трекер месячных бонусов  
✅ **Team Growth Charts** - Графики роста команды (участники и продажи)  
✅ **Invite Functionality** - Функционал приглашения с QR-кодом  

## Компоненты

### MLMDashboardScreen
Главный экран дашборда с табами: Overview, Network, Leaderboard.

### NetworkTree
Иерархическое дерево сети:
- Раскрывающиеся узлы
- Отображение рангов и статусов
- Информация о доходах
- Автоматическое раскрытие первых уровней
- Оптимизация для больших сетей

### TeamStats
Статистика команды:
- Общее количество участников
- Активные участники
- Продажи и заработок
- Рост команды
- Индикаторы в виде карточек

### CommissionBreakdown
Разбивка комиссий:
- Столбчатая диаграмма по уровням
- Детальная информация о каждом уровне
- Процент комиссий
- Количество участников на каждом уровне

### RecruitmentProgress
Прогресс рекрутинга:
- Круговая диаграмма прогресса
- Линейный прогресс-бар
- Информация о следующем бонусе
- Дедлайны и цели

### Leaderboard
Таблица лидеров:
- Ранги с иконками для топ-3
- Разные метрики (продажи, рекруты, заработок, размер команды)
- Индикатор текущего пользователя
- Изменения позиций

### MonthlyBonusTracker
Трекер месячных бонусов:
- Горизонтальный скролл по месяцам
- Статусы (достигнут, в процессе, пропущен)
- Прогресс и цели
- Суммы бонусов

### TeamGrowthChart
Графики роста команды:
- Линейные графики с SVG
- Графики участников и продаж
- Сетка и подписи
- Индикаторы роста

### InviteComponent
Функционал приглашения:
- Пригласительный код
- Ссылка для приглашения
- QR-код (требует библиотеку)
- Шаблон сообщения для шаринга
- Копирование и шаринг

## Использование

```tsx
import { MLMDashboardScreen } from './src/screens/MLMDashboardScreen';

<MLMDashboardScreen
  currentUserId="user-1"
  onMemberPress={(member) => {
    // Navigate to member details
  }}
/>
```

## Типы данных

```typescript
interface MLMMember {
  id: string;
  name: string;
  level: number;
  status: 'active' | 'inactive' | 'pending';
  personalSales: number;
  teamSales: number;
  directReferrals: number;
  totalDownline: number;
  monthlyEarnings: number;
  rank: string;
  children?: MLMMember[];
}

interface NetworkStats {
  totalMembers: number;
  activeMembers: number;
  monthlyTeamSales: number;
  monthlyEarnings: number;
  commissionByLevel: CommissionLevel[];
  growthRate: number;
}
```

## Визуализация данных

### React Native SVG
Все графики используют `react-native-svg`:
- Столбчатые диаграммы
- Круговые прогресс-индикаторы
- Линейные графики роста
- Кастомные иконки и элементы

### Оптимизация производительности
- Виртуализация больших списков
- Ленивое раскрытие узлов дерева
- Мемоизация вычислений
- Ограничение глубины отображения дерева

## Зависимости

```bash
npm install react-native-svg
```

Для QR-кодов (опционально):
```bash
npm install react-native-qrcode-svg
```

## Структура дерева

Дерево сети оптимизировано для:
- Больших сетей (1000+ участников)
- Многоуровневых структур (до 10+ уровней)
- Быстрого отклика при раскрытии/сворачивании
- Эффективной прокрутки

### Конфигурация
```typescript
<NetworkTree
  rootMember={rootMember}
  maxDepth={5} // Ограничение глубины отображения
  onMemberPress={handleMemberPress}
/>
```

## Метрики лидерборда

Поддерживаемые метрики:
- **Sales** - Продажи
- **Referrals** - Рекруты
- **Earnings** - Заработок
- **Team Size** - Размер команды

## Производительность

- **Lazy Loading** - Ленивая загрузка узлов дерева
- **Virtualization** - Виртуализация списков
- **Memoization** - Кэширование вычислений
- **Debouncing** - Ограничение частоты обновлений

## Кастомизация

Все компоненты используют Tailwind CSS и могут быть легко настроены:
- Цвета рангов
- Размеры графиков
- Стили карточек
- Анимации
