# ⚠️ ВАЖНО: Перезапустите серверы!

## Для применения изменений в MLM:

### 1. Остановите серверы:
Закройте окна PowerShell где запущены серверы

### 2. Запустите снова:

#### Backend:
```powershell
cd "Z:\App RBT\backend"
npm run start:dev
```

#### Frontend:
```powershell
cd "Z:\App RBT\web-admin"
npm run dev
```

### 3. Или используйте скрипт:
```powershell
.\START_SERVERS.ps1
```

---

После перезапуска MLM страница будет работать без ошибок!
