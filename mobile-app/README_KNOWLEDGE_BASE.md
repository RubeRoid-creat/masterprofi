# Knowledge Base Screen

Экран базы знаний для ремонта с поиском, руководствами, видео-уроками и всеми инструментами для мастеров.

## Особенности

✅ **Searchable Appliance Manuals** - Поиск по руководствам для приборов  
✅ **Troubleshooting Guides** - Пошаговые руководства по устранению неполадок  
✅ **Video Tutorials** - Видео-уроки с превью  
✅ **Parts Compatibility Database** - База совместимости запчастей  
✅ **Repair Time Estimations** - Оценка времени ремонта  
✅ **Common Issues Solutions** - Решения распространенных проблем  
✅ **Offline Download Capability** - Возможность офлайн-скачивания  
✅ **Favorite Articles** - Избранные статьи  

## Компоненты

### KnowledgeBaseScreen
Главный экран базы знаний с табами для различных типов контента.

### SearchBar
Поисковая строка:
- Поиск по заголовкам, описаниям, тегам
- Кнопка фильтров
- Индикатор активных фильтров

### ArticleCard
Карточка статьи:
- Заголовок и описание
- Категория и иконка
- Типы приборов
- Сложность и время
- Статистика просмотров/лайков
- Кнопки скачивания и избранного

### ArticleViewer
Просмотр статьи:
- Полный контент статьи
- Вложения (PDF, изображения, видео)
- Теги и типы приборов
- Метаданные
- Кнопки действий

### CategoryFilters
Фильтры по категориям:
- Manuals (Руководства)
- Troubleshooting (Устранение неполадок)
- Tutorials (Уроки)
- Parts (Запчасти)
- Common Issues (Распространенные проблемы)
- Repair Guides (Руководства по ремонту)

### TroubleshootingGuide
Руководство по устранению неполадок:
- Симптомы проблемы
- Пошаговые решения
- Раскрывающиеся шаги
- Необходимые инструменты и запчасти
- Оценка сложности и времени

### VideoTutorials
Видео-уроки:
- Горизонтальный скролл
- Превью с длительностью
- Статистика просмотров
- Модальное окно просмотра

### PartsCompatibility
База совместимости запчастей:
- Поиск по номеру запчасти
- Список совместимых запчастей
- Процент совместимости
- Альтернативные запчасти
- Несовместимые запчасти

### RepairEstimation
Оценка времени ремонта:
- Ввод типа прибора и проблемы
- Оценка времени (мин/макс/среднее)
- Оценка сложности
- Необходимые запчасти
- Оценка стоимости

### CommonIssues
Распространенные проблемы:
- Список частых проблем
- Процент частоты
- Быстрые решения
- Ссылки на полные решения

### OfflineDownloads
Офлайн-скачивания:
- Список скачанных статей
- Общий размер
- Удаление скачанных статей

### FavoritesSection
Избранные статьи:
- Все избранные статьи
- Быстрый доступ
- Удаление из избранного

## Использование

```tsx
import { KnowledgeBaseScreen } from './src/screens/KnowledgeBaseScreen';

<KnowledgeBaseScreen userId="user-1" />
```

## Типы данных

```typescript
interface KnowledgeArticle {
  id: string;
  title: string;
  description: string;
  content: string;
  category: ArticleCategory;
  tags: string[];
  applianceTypes: string[];
  difficulty: 'easy' | 'medium' | 'hard';
  estimatedTime: number;
  views: number;
  likes: number;
  isFavorite?: boolean;
  isDownloaded?: boolean;
  attachments?: Attachment[];
}
```

## Поиск и фильтрация

### Поиск
- По заголовку
- По описанию
- По тегам
- По типу прибора

### Фильтры
- Категории статей
- Типы приборов
- Сложность
- Время выполнения
- Наличие видео/вложений

## Офлайн функциональность

### Скачивание
- Скачивание статей для офлайн доступа
- Скачивание вложений (PDF, видео)
- Управление скачанными файлами
- Освобождение места

### Хранение
- Локальное хранение в AsyncStorage
- Кэширование изображений
- Офлайн поиск

## Видео-уроки

Функции:
- Просмотр превью
- Отображение длительности
- Статистика просмотров
- Полноэкранный просмотр
- Интеграция с видеоплеером

## База совместимости запчастей

Возможности:
- Поиск по номеру запчасти
- Процент совместимости
- Альтернативные варианты
- Предупреждения о несовместимости
- Заметки о совместимости

## Оценка времени ремонта

Параметры:
- Тип прибора
- Описание проблемы
- Марка и модель (опционально)

Результат:
- Диапазон времени (мин-макс)
- Среднее время
- Сложность
- Необходимые запчасти
- Оценка стоимости

## Производительность

- Оптимизированный поиск
- Ленивая загрузка контента
- Кэширование результатов
- Виртуализация списков
- Оптимизация изображений
