# Master Profile Screen

Полнофункциональный экран профиля мастера с управлением всей информацией, навыками, портфолио и доступностью.

## Особенности

✅ **Personal Information Editor** - Редактор личной информации с аватаром  
✅ **Skills & Specialties Management** - Управление навыками и специализациями  
✅ **Service Area Configuration** - Конфигурация зоны обслуживания  
✅ **Portfolio Gallery** - Галерея портфолио с добавлением фото  
✅ **Rating & Reviews Display** - Отображение рейтинга и отзывов  
✅ **Availability Calendar** - Календарь доступности с расписанием  
✅ **Statistics** - Статистика (завершенные заказы, рейтинг, время отклика)  
✅ **Certificate Display** - Отображение сертификатов  

## Компоненты

### MasterProfileScreen
Главный экран профиля, объединяющий все секции профиля.

### PersonalInfoEditor
Редактор личной информации:
- Имя, фамилия
- Email, телефон
- Город, адрес
- Биография
- Загрузка аватара
- Режим редактирования

### SkillsManager
Управление навыками:
- Добавление/редактирование/удаление навыков
- Выбор типа прибора
- Уровень навыка (beginner, intermediate, advanced, expert)
- Годы опыта
- Сертификации

### ServiceAreaConfig
Конфигурация зоны обслуживания:
- Центр зоны (координаты и адрес)
- Радиус обслуживания
- Использование текущего местоположения
- Визуализация на карте (placeholder)

### PortfolioGallery
Галерея портфолио:
- Горизонтальный скролл фотографий
- Добавление новых фото
- Удаление фото
- Полноэкранный просмотр
- Подписи к фото

### RatingReviews
Рейтинг и отзывы:
- Общий рейтинг со звездами
- Разбивка рейтинга по звездам
- Список последних отзывов
- Ответы на отзывы
- Аватары клиентов

### AvailabilityCalendar
Календарь доступности:
- Расписание по дням недели
- Включение/выключение дней
- Временные слоты для каждого дня
- Добавление/удаление временных слот
- Статус доступности

### Statistics
Статистика мастера:
- Завершенные заказы
- Общий заработок
- Средний рейтинг
- Время отклика
- Процент завершения
- Процент своевременности

### CertificatesDisplay
Отображение сертификатов:
- Горизонтальный скролл сертификатов
- Изображения сертификатов
- Информация о сертификате
- Проверка истечения срока
- Ссылка на верификацию

## Использование

```tsx
import { MasterProfileScreen } from './src/screens/MasterProfileScreen';

<MasterProfileScreen
  profileId="master-1"
  onProfileUpdate={(profile) => {
    // Handle profile update
  }}
/>
```

## Типы данных

```typescript
interface MasterProfile {
  id: string;
  personalInfo: PersonalInfo;
  skills: Skill[];
  specialties: string[];
  serviceArea: ServiceArea;
  portfolio: PortfolioItem[];
  rating: Rating;
  reviews: Review[];
  availability: Availability;
  statistics: Statistics;
  certificates: Certificate[];
}
```

## Функции редактирования

### Личная информация
- Загрузка аватара из галереи
- Редактирование всех полей
- Сохранение изменений
- Отмена редактирования

### Навыки
- Добавление новых навыков
- Редактирование существующих
- Удаление навыков
- Выбор уровня и типа прибора

### Зона обслуживания
- Установка центра зоны
- Настройка радиуса
- Использование GPS
- Сохранение конфигурации

### Портфолио
- Загрузка фотографий
- Удаление фотографий
- Просмотр в полном размере
- Метаданные фото

### Доступность
- Настройка по дням недели
- Временные слоты
- Исключения (holidays, vacations)
- Общий статус доступности

## Производительность

- Ленивая загрузка изображений
- Оптимизированная прокрутка
- Кэширование данных
- Эффективные обновления UI

## Интеграция API

Все компоненты готовы к интеграции с API:
- `GET /api/masters/:id/profile` - Получение профиля
- `PUT /api/masters/:id/profile` - Обновление профиля
- `POST /api/masters/:id/portfolio` - Добавление в портфолио
- `PUT /api/masters/:id/availability` - Обновление доступности
