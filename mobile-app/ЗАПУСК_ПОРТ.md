# Запуск на другом порту

## Порт 8081 занят

Если порт 8081 занят, используйте другой порт:

```powershell
cd "Z:\App RBT\mobile-app"
npx expo start --port 8082
```

## Остановка процесса на порту 8081 (опционально)

Если хотите освободить порт 8081:

```powershell
# Найти процесс на порту 8081
netstat -ano | findstr :8081

# Остановить процесс (замените PID на номер процесса)
taskkill /PID <PID> /F
```

## Автоматический выбор порта

Expo автоматически предложит использовать другой порт, просто согласитесь (нажмите Enter или введите `y`).

## Быстрый запуск

```powershell
cd "Z:\App RBT\mobile-app"
npm start
```

Если порт занят, используйте:

```powershell
npx expo start --port 8082
```

Приложение запущено на порту 8082 в фоновом режиме. Проверьте терминал или откройте новый терминал и запустите команду заново для интерактивного режима.
